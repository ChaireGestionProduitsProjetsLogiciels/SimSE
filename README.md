# SimSE – Philanthropic Research Chair in Software Product and Project Management


This repository contains the SimSE educational games developed by the  
**Philanthropic Chair in Product, Project and Software Management – ÉTS**.

Each game is organized in a dedicated folder with a common structure to
facilitate access to executables, documentation, and related resources.

## Repository structure

```

Game 1 - Early Sizing Game/
├── downloads/
├── docs/
├── resources/
├── external-content/

Game 2 - Productivity Game/
├── downloads/
├── docs/
├── resources/
├── external-content/

Game 3 - Aposteriori Estimation/
├── downloads/
├── docs/
├── resources/
├── external-content/

Game 4 - Statistical Analysis/
├── downloads/
├── docs/
├── resources/
├── external-content/

```

## Folder description

- **downloads/**  
  Contains the downloadable versions of the game (e.g. Windows `.exe`, Java `.jar`), typically packaged as archives for distribution.

- **docs/**  
  Contains the game documentation, such as the Player’s Manual and the Instructor’s Manual.

- **resources/**  
  Contains any additional resources required to run or use the game properly.

- **external-content/**  
  Contains or references external content used by the game (e.g. spreadsheets, HTML files, or other supporting materials).
