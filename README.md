# SimSE – Philanthropic Research Chair in Software Product and Project Management


This repository contains the SimSE educational games developed by the  
**Philanthropic Chair in Product, Project and Software Management – ÉTS**.

Each game is organized in a dedicated folder with a common structure to
facilitate access to executables, documentation, and related resources.

## Repository structure

```

Game 1 - Early Sizing Game/
├── downloads/
├── docs/
├── resources/
├── external-content/

Game 2 - Productivity Game/
├── downloads/
├── docs/
├── resources/
├── external-content/

Game 3 - Aposteriori Estimation/
├── downloads/
├── docs/
├── resources/
├── external-content/

Game 4 - Statistical Analysis/
├── downloads/
├── docs/
├── resources/
├── external-content/

```

## Notes
- Downloadable games are provided as archives when applicable.
- Documentation is provided in PDF format.
- External content may include additional files or links required by a game.
