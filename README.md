# SimSE – Philanthropic Research Chair in Software Product and Project Management

This repository hosts the SimSE educational games developed by the **Philanthropic Chair in Product, Project and Software Management (ÉTS)**.

It provides:
- **Game packages** for students and instructors (Windows `.exe` and Java `.jar`, distributed as `.zip`)
- **Documentation** (Player’s Manual and Instructor’s Manual)
- **Resources** required to run specific games (if applicable)

## Games available

- Downloads:

## How to download
- Go to the relevant game folder under `downloads/` and download the `.zip`.
- Unzip locally, then run:
  - Windows: the `.exe`
  - Java: the `.jar` (requires Java installed)
